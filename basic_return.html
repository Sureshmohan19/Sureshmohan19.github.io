Simple Returns

In finance, the simple return of an asset (or an investment) can be calculated using the following mathematical formula,

r t = p final_value - p initial_value p initial_value

Everyone will be familiar with this calculation. The method will also help calculate the investors' wealth over a certain period in an investment. For example, let's say that I invest $100 in any stock and if my investment grows to $150 after a certain time, my return on investment will be 50%.

r t = 150 - 100 100

r t = 0.5 x 100 = 50%

Straight forward calculation as it looks. The method is the same for calculating the return of an asset. Now we will see the log return and its calculations.
